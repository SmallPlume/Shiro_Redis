# SpringMVC_Shiro_Redis
主要使用SpringMVC、shiro、redis等写个小的权限管理demo
#配置 版本 ：
  - spring:4.1.7.RELEASE;
  - shiro:1.2.5;
  - mybatis:3.3.0;
  - druid:1.0.25;
  - redis:2.8.0;
  
#  项目URL：http://localhost:8080/SpringMVC_Shiro_Redis/login
